# 1.1.3.  `YANG`

以下术语在[[RFC7950](https://tools.ietf.org/html/rfc7950)]中定义：

- 行动(`action`)

- 容器(`container`)

- 数据节点(`data note`)

- 关键的叶子(`key leaf`)

- 叶子(`leaf`)

- 叶列表(`leaf-list`)

- 列表(`list`)

- 强制节点(`mandatory node`)

- 按用户排序(`ordered-by user`)

- 存在容器(`presence container`)

-  RPC操作(`PRC operation`)

- 顶级数据节点(`top-level data note`)
